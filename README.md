

# reactList

A simple web app practice making lists as React components.

## how

### install

```
git clone https://github.com/rahmona-henry/reactList
cd
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
